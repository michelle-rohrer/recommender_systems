Recommender Systems 

Projektbeschreibung
-------------------
Dieses Projekt implementiert verschiedene Recommender Systems (Empfehlungssysteme) 
basierend auf dem MovieLens Datensatz. Es werden verschiedene Algorithmen implementiert,
evaluiert und optimiert, um Filmempfehlungen für Nutzer zu generieren.

Dataset
-------
Das Projekt verwendet den MovieLens ml-latest-small Datensatz, der sich im Ordner 
`data/` befindet. Der Datensatz enthält:
- 100836 Ratings von 610 Nutzern
- 9742 Filme
- 3683 Tag-Anwendungen
- Zeitraum: März 1996 bis September 2018

Weitere Details zum Datensatz finden Sie in `data/README.txt`.

Projektstruktur
---------------
- data/                    : MovieLens Datensatz (ratings.csv, movies.csv, links.csv, tags.csv)
- RSY_MiniChallenge_HS25.ipynb : Hauptnotebook mit Implementierung und Analyse
- vorbereitung.txt         : Theoretische Grundlagen zu Recommender Systems
- fragen.txt              : Checkliste der behandelten Themen
- readme.txt              : Diese Datei
